# JavaFX Quiz Application

This is a simple and interactive quiz application developed using JavaFX. It presents a set of multiple-choice questions, keeps track of the score, implements a countdown timer, and displays the final results with visual feedback.

## Features

- Multiple-choice quiz with 5 questions  
- Countdown timer (60 seconds)  
- Progress indicators for correct and incorrect answers  
- Result screen with total score  
- Scene transitions using FXML  
- Questions related to Java programming

## Technologies Used

- Java (version 11 or higher)  
- JavaFX  
- FXML  
- CSS for styling  
- Java Timer and Platform APIs
